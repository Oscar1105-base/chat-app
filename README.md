## 事前導覽

### 請先用npm安裝以下套件
- 後端 Node.JS
```
    npm install
```
- 前端 React + firebase 連接 
```
    npm install react-router-dom react-toastify firebase
```
- 開啟本地伺服器
```
    npm run dev
```
